# 中文惯用语教育智能体 - 网页版

轻量级、无需后端的在线学习平台，帮助国际学习者掌握中文惯用语。

## 📁 文件结构

```
web-app/
├── index.html        # 主网页（入口文件）
├── styles.css        # 样式表（深色/浅色主题）
├── app.js            # 应用逻辑（完整功能实现）
├── idioms-db.js      # 知识库数据（9个惯用语 + 练习题）
└── README.md         # 本说明文档
```

## 🚀 快速开始

### 方式1：本地打开（最简单）
1. 找到 `index.html` 文件
2. 用浏览器打开（双击或右键 → 用浏览器打开）
3. 即可使用，无需任何配置或服务器

### 方式2：GitHub Pages 托管（可分享）
1. 上传所有文件到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 获得可分享的网址

### 方式3：本地服务器（推荐）
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server

# 然后在浏览器打开
http://localhost:8000
```

## ✨ 核心功能

### 1️⃣ 惯用语学习
- **9个精选惯用语**：分动物类、数字类、身体类等
- **按HSK等级分类**：初级、中级、高级三个水平
- **详细讲解**：含义、拼音、出处、使用场景、文化背景
- **丰富例句**：每个水平都有对应的例句示例

**已包含的惯用语：**
1. 狐假虎威 - 仗势欺人
2. 卧虎藏龙 - 隐藏的人才
3. 一鸣惊人 - 初出茅庐就成名
4. 十年磨剑 - 长期打磨
5. 千锤百炼 - 反复改进
6. 心如刀割 - 极度伤心
7. 眼高手低 - 高要求低能力
8. 画龙点睛 - 关键一笔
9. 南辕北辙 - 方向错误

### 2️⃣ 四种练习题型
- **填空题**：巩固词汇理解
- **选择题**：辨析近似成语
- **改错题**：学习常见用法错误
- **情景题**：实际应用场景

每道题都有详细的解析和正确答案说明。

### 3️⃣ 学习进度追踪
- ✅ 标记已学的惯用语
- 📊 查看练习统计（正确率、总题数等）
- 📈 实时更新学习进度
- 💾 自动保存本地，永不丢失

### 4️⃣ 交互设计
- 🔍 **搜索功能**：快速查找惯用语
- 🌙 **深色/浅色主题**：保护眼睛，灵活切换
- 📱 **响应式设计**：完美支持手机、平板、电脑
- ⚡ **快速加载**：无依赖，极速运行

## 🎯 使用流程

### 新用户入门
```
首页 → 开始学习 → 浏览所有惯用语 → 选择感兴趣的 → 查看详细讲解
  ↓                                                     ↓
  标记为已学 → 练习中心 → 选择题型 → 完成练习 → 查看统计
```

### 学习建议
1. **循序渐进**：先学初级，再学中级，最后学高级
2. **深入理解**：不仅读讲解，还要理解文化背景
3. **反复练习**：完成各种题型的练习，巩固知识
4. **实际应用**：在日常对话中尝试使用这些成语

## 💾 数据存储

- **存储位置**：浏览器 LocalStorage
- **数据内容**：学习进度、已标记的惯用语、练习记录
- **数据隐私**：完全本地存储，不上传任何数据
- **清除数据**：可在进度页面点击"重置进度"按钮

> **提示**：如果清空浏览器数据或使用无痕模式，学习记录会丢失。建议定期保存重要数据。

## 🔧 扩展和定制

### 添加新的惯用语
编辑 `idioms-db.js`，在 `idiomsDatabase` 数组中添加新项：

```javascript
{
  id: 10,
  name: "新的惯用语",
  pinyin: "xīn de guànyòng yǔ",
  level: "中级",
  hskLevel: ["中级"],
  meaning: "含义说明",
  category: "分类",
  // ... 其他字段
}
```

### 自定义样式
编辑 `styles.css` 中的 CSS 变量：

```css
:root {
  --primary-color: #4a90e2;      /* 主色调 */
  --secondary-color: #f5a623;    /* 辅助色 */
  --success-color: #7ed321;      /* 成功色 */
  /* ... 其他颜色 */
}
```

### 添加新的练习题
在 `idioms-db.js` 的 `exerciseDatabase` 中添加新题目。

## ❓ 常见问题

**Q: 我的学习记录被删除了？**
A: 检查是否清空了浏览器缓存或 Cookie。建议定期导出数据备份。

**Q: 可以在多个设备同步学习记录吗？**
A: 当前版本使用本地存储，不支持跨设备同步。可考虑升级到 MCP 服务器版本获得云同步功能。

**Q: 可以添加更多的惯用语和题目吗？**
A: 当然可以！编辑 `idioms-db.js` 按照现有格式添加新内容即可。

**Q: 如何离线使用？**
A: 将所有文件保存到本地，用浏览器打开 `index.html` 即可离线使用。数据存储在本地，无需网络连接。

**Q: 手机上使用会不会卡？**
A: 不会。应用设计轻量级，内存占用很低，即使在旧手机上也能流畅运行。
